## Requirements

For development, you will only need Node.js installed on your environement.

## Install

    $ git clone https://github.com/mrtins/react-context-counter-example.git
    $ cd react-context-counter-example
    $ npm install

## Start & watch

    $ npm start

## Simple build for production

    $ npm run build
